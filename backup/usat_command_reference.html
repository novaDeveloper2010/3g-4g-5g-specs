1. Proactive Commands

Commands the UICC sends to the terminal. This page focuses on the rows most useful for a compact browser reference.

UI Checkbox ETSI Command Clause(s) Applies Source Description

Get Inkey GET INKEY 6.4.2 3G 3GPP TS 31.111 R7 (USAT) UICC requests a single character (digit, character or Yes/No) keystroke from the user.
Get Input GET INPUT 6.4.3 3G 3GPP TS 31.111 R7 (USAT) UICC requests the ME to prompt the user for a string of characters, with min/max length and echo/hide options.
Play Tone PLAY TONE 6.4.5 3G 3GPP TS 31.111 R7 (USAT) UICC instructs the ME to play an audible tone for a specified duration.
Refresh REFRESH 6.4.7 3G 3GPP TS 31.111 R7 (USAT) UICC notifies the ME that file contents or configuration changed; triggers cache invalidation, reset or full re-initialisation.
Set Up Menu SET UP MENU 6.4.8 3G 3GPP TS 31.111 R7 (USAT) UICC supplies the ME with the main SIM Toolkit menu of items to present to the user.
Select Item SELECT ITEM 6.4.9 3G 3GPP TS 31.111 R7 (USAT) UICC asks the ME to present a list of items and return the user's selection.
