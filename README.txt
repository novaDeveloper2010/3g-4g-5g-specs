ETSI CAT / 3GPP USAT spec archive, bucketed by technology era

The 3g/, 4g/, and 5g/ folders are kept on purpose.
They are era buckets, not strict generation-pure documents.

Each bucket holds the latest ETSI TS 102 223 (CAT)
and 3GPP TS 31.111 (USAT) release available for that era:

- 3g/ — 3G-era snapshot (Release 7)
- 4g/ — 4G-era snapshot (Release 14)
- 5g/ — 5G-era snapshot (Release 18)

Use the HTML command reference to browse the command list
for each era and the matching PDFs in each folder.
