ETSI CAT / 3GPP USAT spec archive, organized by release stack

The 3g/, 4g/, and 5g/ folders are kept on purpose.
Each folder is one distinct 3GPP release, and later releases build on earlier ones:

- 3g/ — Release 7, the latest 3G-era baseline
- 4g/ — Release 14, which extends Release 7
- 5g/ — Release 18, which extends Release 14

Use the HTML command reference to browse the command list
for each release and the matching PDFs in each folder.
